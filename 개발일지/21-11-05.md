# 2021-11-05 진행사항

### 1. api 추가

##### 1) 내 동네 추가

내 동네 설정에서 설정된 동네가 2개 미만일 때 새로운 동네를 추가할 수 있는 api

> POST
> /address/:townId

##### 2) 내 동네 삭제

내 동네 설정에서 설정된 동네가 1개 이상일 때 동네를 삭제할 수 있는 api

> PATCH
> /address/:townId

##### 3) 내 동네 변경

내 동네 설정에서 설정된 동네가 2개일 때 동네를 변경할 수 있는 api

> PATCH
> /address/change/:townId
